## Production Notes

### Camera/Screen Recording Tips
1. Record at 4K resolution (3840x2160) for crisp details
2. Use screen recording software (OBS, Screenflow, ScreenFlow)
3. Zoom into important code sections for readability
4. Show actual terminal output for authenticity
5. Include cursor movements for clarity

### Voice-Over Recording
1. Record in quiet environment
2. Use professional microphone for clarity
3. Speak clearly at natural pace
4. Emphasize key points with intonation
5. Add enthusiasm when discussing features

### Post-Production
1. Add subtitle overlays for accessibility
2. Sync narration precisely with visuals
3. Add sound effects for transitions (optional but effective)
4. Include background music at appropriate volume
5. Add watermark or branding at end

### File Format
- Export as MP4 (H.264 codec)
- 1080p minimum, 4K preferred
- 60 FPS for smooth motion
- Include subtitles/captions in English
